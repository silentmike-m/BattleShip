# BattleShip
## Req

* .NET 6
* Project Tye
* Docker

## Run application

### .NET CLI
Application is available at http://localhost:30000

```console
cd .\src\BattleShip.WebApi\
dotnet run
```

### Docker
Application is available at http://localhost:30000

```console
cd .\src\
docker-compose up
```

### Tye
Application is available at http://localhost:8000

```console
cd .\src\
tye run
```
